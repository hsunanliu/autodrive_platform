## 📝 文檔維護

文檔應該保持更新，反映最新的系統狀態。

### 更新原則
- 新功能完成時更新 PROJECT_STATUS.md
- 架構變更時更新 architecture.md
- 新增配置時在 setup/ 目錄添加文檔
- 使用流程變更時更新使用者指南

1. 開 Issue 說明問題

3. 在團隊會議中提出
